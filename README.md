**Adjustable DC-DC Buck converter with LM2596-ADJ**

A versatile Adjustable Buck Converter Module built around the LM2596-ADJ switching regulator.
It steps down higher DC voltages to a stable lower voltage using high-efficiency switching technology, ideal for powering microcontrollers, sensors, and embedded systems.

<img width="1674" height="944" alt="Bottom-img" src="https://github.com/user-attachments/assets/ef93ab0f-79d1-424b-b406-bed6f4a34763" />

**Electrical Specifications**

| Parameter             | Value / Description               |
| --------------------- | --------------------------------- |
| Input Voltage (VIN)   | 4.5 V – 40 V DC                   |
| Output Voltage (VOUT) | Adjustable (1.2 V – 37 V DC)      |
| Max Output Current    | 2 A (typical), 3 A peak           |
| Efficiency            | Up to 92% (depending on VIN/VOUT) |
| Regulator IC          | LM2596-ADJ Buck Converter         |
| Switching Frequency   | 150 kHz                           |
| Output Adjust         | Multi-turn potentiometer          |
| PCB Type              | 2-Layer, designed in KiCad 9.0.1  |

**Features**

High-efficiency LM2596-ADJ step-down converter
Stable adjustable output ranging from 1.2 V to 37 V
On-board multi-turn potentiometer for precise voltage control
Input/output filtering capacitors for clean regulation
Works with battery packs, adapters, solar setups, and custom electronics
Compact PCB layout designed in KiCad 9.0.1
